# AI Tutor Workflow 🤖

An AI-powered tutor workflow built using n8n + Gemini AI.

## Features
- Intelligent AI responses
- Memory-based conversations
- Google Sheets integration
- Web search functionality
- Automated workflow system

## Technologies Used
- n8n
- Gemini AI
- Google Sheets
- AI Automation

## Purpose
This project explores how AI agents can create smarter learning systems and automate educational support workflows.
